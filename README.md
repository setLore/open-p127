# open127

## Description

A universal program, with the purpose of downgrading and upgrading GTA 5 to versions `1.24` / `1.27` / `1.29`

* Works on Linux and Windows

## Getting started

### Dependencies

* Python 3
* The [customtkinter](https://pypi.org/project/customtkinter/) package(installed by running `pip3 install customtkinter` in your terminal)

### Installing

* Download the p127.py file
* Put the file in your GTA 5 installation directory

### Running the program

* Open your terminal
* Go to the directory of the program and GTA 5 using the `cd` command
* Run `python3 p127.py`
